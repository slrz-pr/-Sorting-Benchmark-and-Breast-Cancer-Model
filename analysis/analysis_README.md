# Sorting & Breast Cancer Model Analysis Summary

## Dataset Reference
- **Original Source:** UCI Breast Cancer Wisconsin (Diagnostic) Dataset
- **URL:** https://archive.ics.uci.edu/ml/datasets/breast+cancer+wisconsin+(diagnostic)

## Summary Findings
* **Sorting Efficiency:** Quick Sort [O(n log n)] outperformed Bubble Sort [O(n^2)] by a massive margin. In bioinformatics, efficient sorting is the backbone of sequence alignment and genome assembly.
* **Regression Analysis:** The linear regression model captured a slope of ~6.28, reflecting the geometric relationship (circumference) of the cell nuclei. It allows us to predict unknown perimeter dimensions from radius measurements.
* **Classification Accuracy:** Our model achieved high accuracy/precision. In clinical bioinformatics, this demonstrates how automated morphological screening can assist in early cancer detection.
* **Biological Clustering:** By selecting 2 clusters, we successfully partitioned samples into two groups that biologically mirror the "Benign" and "Malignant" tumor subtypes based on nucleus irregularity.
